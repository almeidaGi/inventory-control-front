Filtrar um produto

Para filtrar um produto, na tela inicial, utilize o campo "Filtre por Nome ou Descrição"


Adicionar um produto

Para adicionar um produto, na tela inicial, clique nos três pontose clique em adicionar produto

Editar um produto

Para editar um produto, na tela inicial, na lista de produtos clique no lapis

Deletar um produto

Para deletar um produto, na tela inicial, na lista de produtos clique na lixeira
